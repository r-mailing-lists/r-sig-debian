# r-sig-debian

Debian ports

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-sig-debian/](https://stat.ethz.ch/pipermail/r-sig-debian/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics
